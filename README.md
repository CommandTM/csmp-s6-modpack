# The Official Modpack For Cat SMP Season 6
This is the modpack you'll need to connect to the sixth season of Cat SMP

## What Is Cat SMP?
If you're asking that question, this modpack probably isn't for you. Cat SMP is a private server between friends. We do not openly accept members. 
This modpack was handmade for our server, but we think it could be a fun collection to possibly start your own server with, so go ahead, try it out!

## Extra Install Steps
This modpack was intended to ship with 3 additional mods, however due to licensing issues you will have to add them yourself. Links to the specifically intended version will be included below, sorry!
